# ロマンジ

フリック入力で日本語の入力を練習するためのウェブアプリです。

## 機能

- iPhone風かな12キー・フリック入力キーボード
- JLPT N5〜N1 単語帳対応
- 学習進捗の記録・PB（自己ベスト）の追跡
- テーマ切替（ダーク/ライト）

## クレジット

### 単語データ

JLPT単語データは [elzup/jlpt-word-list](https://github.com/elzup/jlpt-word-list) を利用しています。
